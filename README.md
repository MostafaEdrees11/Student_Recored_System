Student Record System is a project used to manage the data of the students in a school.

Project has two modes:

* Admin Mode
        
        Admin has features like:
        
                * Add student record
                * Remove student record
                * View student record
                * View all records
                * Edit admin password
                * Edit student grade

* Student Mode
        
        Student has features like:
        
                * View your record.
                * Edit your password.
                * Edit your name
